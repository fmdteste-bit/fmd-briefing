Como criar o Worker em 3 minutos
  1. Acesse dash.cloudflare.com e faça login (conta gratuita)
  2. Menu lateral → Workers & Pages → Create
  3. Escolha Create Worker → clique em Deploy
  4. Clique em Edit code, apague tudo e cole o conteúdo do arquivo worker.js
  5. Clique em Deploy — copie a URL gerada (ex: fmd-briefing.seu.workers.dev)
  6. Cole essa URL no campo acima e configure sua chave API
